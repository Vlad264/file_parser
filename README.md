# file_parser
Simple file parser to find number of words or count checksum

Synopsis
        parse [-h]
              [-f] [-m]
              [-f] [-m] [-v]

Program Options:
  -h [ --help ]          Display help menu.
  -f [ --file ] arg      User-specified input file.
  -m [ --mode ] arg      User-specified program mode.
  -v [ --word ] arg      User-specified program word.
